hexo+github搭建个人博客

Posted on 2018-10-19

一、准备github repository

1、新建一个仓库名称 repository name 设置为：用户名.github.io（eg：essencepei.github.io）
2、必须选中Initialize this repository with a README

二、准备hexo

1、在本地新建一个空文件夹
2、进入文件夹所在目录，npm install hexo -g安装hexo
3、hexo -v 验证是否安装成功
4、hexo init，看到Start blogging with Hexo！表示成功
5、npm install 安装nodejs modules
6、hexo g
7、hexo s本地预览，看到hexo页表示成功

三、连接hexo 和 github page

1、文件夹下启动git命令窗口
2、git config –global user.name “” //github用户名
git config –global user.email “” //github登录邮箱
ssh -T git@github.com 测试添加ssh是否成功，成功表示可以发布到github上了
3、修改_config.yml
deploy:
type: git
repository: https://github.com/essencepei/essencepei.github.io.git
branch: master

四、新建文章

hexo new post “文章名称”

五、本地预览

hexo clean
hexo s
预览地址：localhost:4000

六、发布到github page

hexo d -g

七、更换主题

1、git clone https://github.com/giscafer/hexo-theme-cafe.git themes/cafe 下载主题源码文件
2、修改博客配置文件 _config.yml 主题属性 theme 为 cafe.
3、cd themes/cafe
git pull
