hexo+github搭建个人博客

准备github repository

新建一个仓库名称 repository name 设置为：用户名.github.io（eg：essencepei.github.io）
必须选中Initialize this repository with a README

准备hexo

在本地新建一个空文件夹

连接hexo 和 github page

更换主题

1、git clone https://github.com/giscafer/hexo-theme-cafe.git themes/cafe 下载主题源码文件
2、修改博客配置文件 _config.yml 主题属性 theme 为 cafe.
3、cd themes/cafe
git pull
